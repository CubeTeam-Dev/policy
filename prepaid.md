# 前払式支払手段に関する表示

本表示は、資金決済に関する法律（資金決済法）第13条に基づき、当チームが発行する「Cubeクレジット（CC）」に関する事項を表示するものです。

| 項目                     | 内容                                                                                         |
| ------------------------ | -------------------------------------------------------------------------------------------- |
| **前払式支払手段の名称** | Cubeクレジット（CC）                                                                         |
| **発行者**               | Cube Team（運営責任者: 飯田 陸）                                                             |
| **支払可能金額等**       | 利用者が保有する未使用クレジット残高の範囲内                                                 |
| **有効期限**             | クレジット付与（発行）日から1年間                                                            |
| **利用できる範囲**       | 本サービス「Cube」内の対象機能（AIモデレーション等の従量課金対象機能）における利用に限ります |
| **未使用残高の確認方法** | ダッシュボードの「クレジット」画面にて随時ご確認いただけます                                 |
| **苦情・相談窓口**       | 公式サポートサーバー、またはメール [support@cubeteam.app](mailto:support@cubeteam.app)       |

## 有効期限について

Cubeクレジットの有効期限は、付与（発行）日から **1年間** です。有効期限を経過した未使用クレジットは失効し、利用できなくなります。

## 利用上の注意

- Cubeクレジットは、本サービス「Cube」内の対象機能でのみ利用できます。現金との交換、他のサービスへの移転、第三者への譲渡はできません。
- クレジットの利用は、ダッシュボードおよびBotを通じて行われます。
- 保有する未使用残高を超える利用はできません。

## 払戻しについて

Cubeクレジットは、その性質上、原則として払戻しを行いません。

ただし、資金決済法第20条に基づき、当チームが前払式支払手段の発行を終了する場合など、法令で定められた場合には、所定の手続きにより未使用残高の払戻しを行うことがあります。この場合の手続きは、本サービス上で別途告知します。

## 発行保証金の供託等について

当チームは、資金決済法の定めるところにより、基準日（毎年3月31日および9月30日）における未使用残高が基準額（1,000万円）を超える場合、法令に従い発行保証金の供託その他必要な保全措置を講じます。
